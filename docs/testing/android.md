# Checklist: Android

## Prostředí

- Zařízení:
- Verze Androidu:
- Test: Chrome PWA / Android build
- Datum testu:

## Chrome / PWA

- [ ] Aplikace se otevře v Chrome.
- [ ] Zobrazí se `1469 otázek v databázi`.
- [ ] Chrome nabídne instalaci aplikace.
- [ ] Aplikace se přidá na plochu.
- [ ] Aplikace se spustí jako samostatné okno.

## Android build

- [ ] Projekt se otevře v Android Studiu.
- [ ] Build proběhne bez chyby.
- [ ] Aplikace se nainstaluje do zařízení/emulátoru.
- [ ] Aplikace se spustí bez bílé obrazovky.

## Ovládání

- [ ] Funguje volba Letoun / Vrtulník / Obojí.
- [ ] Funguje výběr kategorií.
- [ ] Odpovědi jsou dobře klikatelné.
- [ ] Mapa otázek je čitelná.
- [ ] Rozhraní nemá vodorovné posouvání.

## Režimy

- [ ] Funguje Učit se.
- [ ] Funguje Test.
- [ ] Funguje Zkouška.
- [ ] Funguje Chyby.

## Offline test

- [ ] Po prvním načtení funguje aplikace offline.
- [ ] Statistiky zůstávají uložené po restartu aplikace.
