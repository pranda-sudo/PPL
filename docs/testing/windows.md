# Checklist: Windows

## Prostředí

- Zařízení:
- Verze Windows:
- Typ buildu: Portable EXE / Instalátor
- Datum testu:

## Instalace a spuštění

- [ ] Portable `.exe` se spustí bez instalace.
- [ ] Instalátor se spustí bez chyby.
- [ ] Instalátor vytvoří zástupce na ploše nebo v nabídce Start.
- [ ] Aplikace se otevře v samostatném okně.
- [ ] Okno lze zvětšit, zmenšit a zavřít.

## Základní funkce

- [ ] Zobrazí se `1469 otázek v databázi`.
- [ ] Funguje volba Letoun / Vrtulník / Obojí.
- [ ] Funguje výběr kategorií.
- [ ] Funguje režim Učit se.
- [ ] Funguje režim Test.
- [ ] Funguje režim Zkouška.
- [ ] Funguje režim Chyby.

## Zkouška

- [ ] Zkouška má 120 otázek.
- [ ] Časovač běží.
- [ ] Vyhodnocení zkoušky zobrazí protokol.
- [ ] Přehled předmětů se počítá správně.

## Ukládání dat

- [ ] Odpovědět na několik otázek.
- [ ] Zavřít aplikaci.
- [ ] Znovu otevřít aplikaci.
- [ ] Statistiky zůstaly uložené.
- [ ] Chybové otázky zůstaly uložené.

## Offline test

- [ ] Spustit aplikaci bez internetu.
- [ ] Otázky jsou dostupné.
- [ ] Odpovídání a statistiky fungují.
