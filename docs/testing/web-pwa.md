# Checklist: Web / PWA

## Prostředí

- Prohlížeč:
- Verze prohlížeče:
- Operační systém:
- URL aplikace:
- Datum testu:

## Načtení aplikace

- [ ] Aplikace se otevře bez chyby.
- [ ] Zobrazí se `1469 otázek v databázi`.
- [ ] Zobrazí se kategorie.
- [ ] Zobrazí se tlačítka Letoun / Vrtulník / Obojí.
- [ ] První otázka má odpovědi.

## Instalace jako PWA

- [ ] Prohlížeč nabízí instalaci aplikace.
- [ ] Aplikace se nainstaluje.
- [ ] Aplikace se spustí jako samostatné okno.
- [ ] Ikona aplikace se zobrazí správně.

## Režim Učit se

- [ ] Lze vybrat Letoun.
- [ ] Lze vybrat Vrtulník.
- [ ] Lze vybrat Obojí.
- [ ] Lze vybrat pouze jeden předmět.
- [ ] Po odpovědi se správná odpověď označí zeleně.
- [ ] Po špatné odpovědi se zvolená odpověď označí červeně.
- [ ] Mapa otázek mění barvy podle výsledku.
- [ ] Tlačítka Předchozí otázka / Další otázka fungují.

## Režim Test

- [ ] Při výběru více předmětů se použije nastavený počet otázek.
- [ ] Při výběru jednoho předmětu se zobrazí celá série daného předmětu.
- [ ] Statistiky v horním panelu se počítají správně.

## Režim Zkouška

- [ ] Zkouška má 120 otázek.
- [ ] Časovač začíná na 4:00:00.
- [ ] Přehled předmětů obsahuje 9 předmětů.
- [ ] Tlačítko Vyhodnotit zkoušku zobrazí výsledkový protokol.
- [ ] Výsledkový protokol ukazuje správně počet správných/chybných odpovědí.
- [ ] Výsledkový protokol ukazuje úspěšnost po předmětech.

## Režim Chyby

- [ ] Po špatné odpovědi se otázka objeví v režimu Chyby.
- [ ] Po označení otázky jako Umím se otázka z chyb odebere.

## Offline test

- [ ] Aplikace se jednou načte online.
- [ ] Po vypnutí internetu se aplikace znovu otevře.
- [ ] Otázky jsou dostupné offline.
- [ ] Statistiky fungují offline.
